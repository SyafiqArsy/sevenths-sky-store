## рҹҡҖ About

**Seventh Sky Store** is a modern, full-stack e-commerce web application designed for a premium streetwear fashion brand. Built with a **headless architecture**, the backend (Laravel REST API) and frontend (Next.js App Router) are fully decoupled, communicating through JSON endpoints.

The platform delivers a seamless shopping experience вҖ” from browsing products with immersive WebGL visual effects, through a smooth checkout process with **Midtrans Snap** payment gateway, to real-time order tracking. An integrated admin dashboard provides complete control over products, categories, orders, and revenue analytics.

> рҹҮ®рҹҮ© Built with the Indonesian market in mind вҖ” prices in Rupiah, Midtrans payment gateway, and localized address format.

## вңЁ Key Features

### рҹӣҚпёҸ Customer Experience
- **Product Catalog** вҖ” Browse products with search and category filtering
- **Product Detail** вҖ” Rich product pages with image galleries and quantity selectors
- **Shopping Cart** вҖ” Add, update, and remove items with real-time cart management
- **Secure Checkout** вҖ” Shipping form with Midtrans Snap payment popup
- **Order History** вҖ” Track order status (pending вҶ’ processing вҶ’ shipped вҶ’ completed)
- **User Authentication** вҖ” Register, login, and token-based session management

### рҹҺЁ Visual & UI
- **WebGL Galaxy Effects** вҖ” Immersive 3D starfield shader on the Collections page (built with OGL + custom GLSL)
- **Parallax Animations** вҖ” Smooth parallax scrolling on the homepage hero
- **Custom Cursor System** вҖ” Interactive cursor with explore and pointer modes
- **Smooth Scrolling** вҖ” Butter-smooth scroll behavior powered by Lenis
- **Framer Motion Animations** вҖ” Fluid page transitions and hover effects
- **Responsive Design** вҖ” Fully responsive across all device sizes

### рҹ”§ Admin Dashboard
- **Revenue Analytics** вҖ” Real-time stats for products, categories, orders, and total revenue
- **Product Management** вҖ” Full CRUD with Cloudinary image upload and stock management
- **Category Management** вҖ” Create, edit, and delete product categories
- **Order Management** вҖ” View all orders, update status with state machine validation
- **Role-Based Access** вҖ” Admin-only routes with middleware protection

### рҹ’і Payment & Orders
- **Midtrans Snap Integration** вҖ” Full payment flow with Snap popup
- **Webhook Handling** вҖ” Server-to-server payment verification (SHA-512 signature)
- **Stock Management** вҖ” Auto-decrement on checkout, restore on payment failure/expiration
- **Order State Machine** вҖ” Enforced status transitions with validation

## рҹ”Ң API Endpoints

### Public

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/register` | Register a new account |
| `POST` | `/api/login` | Login and receive token |
| `GET` | `/api/products` | List active products (search & filter) |
| `GET` | `/api/products/{slug}` | Get product by slug |
| `GET` | `/api/categories` | List all categories |
| `GET` | `/api/categories/{slug}` | Get category by slug |
| `POST` | `/api/midtrans/notification` | Payment webhook |

### Authenticated (requires `Bearer` token)

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/api/me` | Get current user profile |
| `POST` | `/api/logout` | Revoke current token |
| `GET` | `/api/cart` | Get user's cart |
| `POST` | `/api/cart/items` | Add item to cart |
| `PUT` | `/api/cart/items/{id}` | Update cart item quantity |
| `DELETE` | `/api/cart/items/{id}` | Remove item from cart |
| `POST` | `/api/checkout` | Create order & get payment token |
| `GET` | `/api/orders` | Get order history |
| `GET` | `/api/orders/{id}` | Get order detail |

### Admin (requires `admin` role)

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/api/admin/dashboard` | Dashboard statistics |
| `CRUD` | `/api/admin/categories/*` | Category management |
| `CRUD` | `/api/admin/products/*` | Product management |
| `GET` | `/api/admin/orders` | List all orders |
| `PUT` | `/api/admin/orders/{id}/status` | Update order status |

## рҹҸҒ Getting Started

### Prerequisites

- **PHP** >= 8.3
- **Composer** >= 2.x
- **Node.js** >= 18.x
- **npm** >= 9.x

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/seventh-sky-store.git
cd seventh-sky-store
```

### 2. Backend Setup

```bash
cd backend

# Install dependencies
composer install

# Create environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Create SQLite database
touch database/database.sqlite

# Run migrations
php artisan migrate

# Seed database (optional)
php artisan db:seed

# Start development server
php artisan serve
```

#### Backend `.env` Configuration

```env
DB_CONNECTION=sqlite

MIDTRANS_SERVER_KEY=your_midtrans_server_key
MIDTRANS_CLIENT_KEY=your_midtrans_client_key
MIDTRANS_IS_PRODUCTION=false

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

### 3. Frontend Setup

```bash
cd frontend

# Install dependencies
npm install

# Create environment file
cp .env.example .env.local

# Start development server
npm run dev
```

#### Frontend `.env.local` Configuration

```env
NEXT_PUBLIC_API_URL=http://localhost:8000/api
NEXT_PUBLIC_MIDTRANS_CLIENT_KEY=your_midtrans_client_key
```

### 4. Access the Application

| URL | Description |
|---|---|
| `http://localhost:3000` | Frontend (Next.js) |
| `http://localhost:8000` | Backend API (Laravel) |
